## Getting Started

```bash
# Clone this project
$ git clone https://github.com/irangarcia/avatar-uploader

# Access
$ cd avatar-uploader

# Install dependencies
$ yarn install

# Run the project
$ yarn start

# To run tests
$ yarn test

# The server will initialize in the <http://localhost:3000>
```
